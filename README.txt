# Realtime Brigadas (Supabase + Render)

1. Rellena `.env` con tus claves de Supabase (`SUPABASE_URL`, `SUPABASE_ANON_KEY`, `SUPABASE_SERVICE_ROLE_KEY`).  
2. `npm i`  
3. `npm start`  
4. Abrir `https://<tu-app>.onrender.com/login.html` para iniciar sesión.  
5. Técnicos: `/` → iniciar trabajo.  
6. Viewer: `/viewer.html`.
